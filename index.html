🔑 Write this down or screenshot it. If you clear browser data or switch phones, open the app → tap "Restore your profile" → enter this ID → your data returns instantly (as long as you've exported a backup below).
Backup & Restore 🔐
Your data lives in your browser. Back up regularly — especially before clearing browser data or switching devices. GitHub Backup saves directly to your repo and is versioned forever.
Storage Used
—
localStorage limit ~5MB · comfortably fits 5–10 years of data
Export = download a .json file of everything · Import = restore from that file (merges intelligently, won't erase new data)
🐙
GitHub Auto-Backup
Saves your data.json directly to your repo — versioned, free, recoverable from any device
🔑 Get a token: github.com → Settings → Developer settings → Personal access tokens → Generate new token (classic) → tick repo scope → copy it here. Token is stored only in your browser.
Migration
Moving to a new phone or browser? Export on your old device, then use Import JSON on the new one. The import is smart — it merges old data with any new entries, never overwrites newer records.
What gets migrated: all habits & completion history · milk calendar · finance entries & categories · shopping lists · cycle data · all settings & preferences
Safe to import on top of existing data — newer entries win on conflict.

Install LifePulse
Works offline · No app store · Feels native
🔵 Android · Chrome
1
Open this page in Chrome browser on your Android phone
2
Tap the ⋮ three-dot menu (top right of Chrome)
3
Look for "Add to Home screen" or "Install app" — it appears only if the page meets Chrome's PWA criteria
✓
If you see it — tap it and confirm. The LifePulse icon will appear on your home screen!
⚠️ Chrome not showing "Add to Home Screen"?
Chrome requires the page to be served over HTTPS to show the install option. Since this is a local HTML file, here's the best workaround:
🌐
Option A — Host it free: Upload this HTML file to GitHub Pages, Netlify Drop, or Vercel. Then open the HTTPS link in Chrome — install button will appear instantly.
📁
Option B — Kiwi Browser: Install Kiwi Browser from Play Store. It supports installing local HTML files as home screen apps via its menu.
🔖
Option C — Bookmark shortcut: In Chrome, open the ⋮ menu → Add to bookmarks. Then create a shortcut: go to your Android home screen → long press → Widgets → Chrome shortcut → pick the bookmark.
🍎 iPhone · Safari
1
Open in Safari (not Chrome — Safari is required on iPhone)
2
Tap the Share ⎋ button at the bottom of Safari
3
Scroll down in the share sheet → tap "Add to Home Screen"
✓
Tap Add — the ⚡ LifePulse icon appears on your home screen and runs like an app!
LifePulse works fully offline once loaded 🔒
